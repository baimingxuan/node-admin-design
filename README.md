# Koa Admin Design

基于 Koa 开发的用于 admin-design 的后台管理系统 API 服务。

## 技术栈

- Koa.js - 轻量级 Node.js Web 框架
- Sequelize - ORM 框架
- MySQL - 数据库
- JWT - 认证
- bcrypt - 密码加密

## 功能特性

- 用户认证与授权
- 用户管理
- 角色管理
- 菜单管理
- 部门管理
- 岗位管理
- 字典管理

## 安装与运行

### 前提条件

- Node.js (>= 12.x)
- MySQL (>= 5.7)
